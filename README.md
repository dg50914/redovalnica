# redovalnica

Paket redovalnica ponuja preprost način upravljanja z ocenami študentov, prikazovanjem redovalnice in izračunavanjem povprečij.

Demo program sprejema tri argumente:

- `--stOcen <n>` določa minimalno število ocen, ki jih študent potrebuje, da se mu izračuna povprečje
- `--minOcena <n>` določa minimalno oceno, ki jo bo redovalnica sprejela
- `--maxOcena <n>` določa maksimalno oceno, ki jo bo redovalnica sprejela
